## What is this?

Every 2 hours, a GitHub Actions workflow runs three evolution engines, advancing the world by 5 generations each. The visualizations run client-side simulations between server ticks so you always see life in motion.

## The Synapse

A living neural network that grows, fires, dreams, and evolves. Neurons carry a 12-gene genome controlling their firing behavior, learning rates, and signal characteristics. Synapses strengthen through Hebbian learning and weaken when unused.
